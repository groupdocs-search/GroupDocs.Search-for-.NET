Immediately after you create the Movie LINQ to SQL Classes, the Object Relational Designer  
appears. You can drag database tables from the Server Explorer window onto the Object  
Relational Designer to create LINQ to SQL Classes that represent particular database tables.  
We need to add the tblMovie database table onto the Object Relational Designer (see Figure  
4).  
Figure 4 Using the Object Relational Designer  
By default, the Object Relational Designer creates a class with the very same name as the  
database table that you drag onto the Designer. However, we dont want to call our class  
tblMovie. Therefore, click the name of the class in the Designer and change the name of the  
class to Movie.  
Finally, remember to click the
Save
button (the picture of the floppy) to save the LINQ to  
SQL Classes. Otherwise, the LINQ to SQL Classes wont be generated by the Object  
Relational Designer.
